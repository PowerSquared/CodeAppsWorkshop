# Module 01 — Get your Code App Running

Before the workshop starts, download the starter files and get your environment ready.

## Steps

### 1. Download the starter files

Clone or download the following repository and save it into a folder on your machine.

> **Tip:** We recommend `C:\code\` — it's easy to reach from the terminal and cloud storage software won't lock your files.

[github.com/PowerSquared/code-apps-plugin-custom](https://github.com/PowerSquared/code-apps-plugin-custom)

```bash
git clone https://github.com/PowerSquared/code-apps-plugin-custom C:\code\code-apps-plugin-custom
```

### 2. Open the folder in VS Code

```bash
code C:\code\code-apps-plugin-custom
```

### 3. Verify your setup

Run the following in a VS Code terminal to confirm everything is installed:

```bash
node --version   # should be v20 or higher
git --version
pac --version
```

Once you see version numbers for all three, you are ready to go. Move on to [Module 02](Module-02.md) when the session starts.
