# Module 02 — Architecture & AI-Assisted Planning

In this module you will use Kilo Code to scaffold a new Code App and then use the PRD and design contract to lay out the app's pages and components with hard-coded data.

## Prompts

### 1. Scaffold the Code App

Open Kilo Code and submit the following prompt:

```
Can you scaffold a new code app in our current directory? Just scaffold it. The display name will be 'Asset Register'.
```

### 2. Design the app

```
In the docs folder there's a PRD and a design contract. Using these, can you create the design and the pages for the app. You can hard code the data for now as we do not have any Dataverse tables scaffolded yet.
```

> **What to expect:** Kilo Code will generate the page structure and components based on the documents. Review the output before moving on — make sure the pages match the design contract.

When you are happy with the scaffolded app, move on to [Module 03](Module-03.md).
